DiarAIAI 日记分析工具核心架构设计文档————————————————————————最后更新：2026-04-201. 项目概述DiarAI 是一款面向个人用户的 AI 日记分析工具，支持 AI 引导写作、情感趋势分析、双向标签体系与个性画像生成。该项目采用纯本地运行方式，数据存储在本地 SQLite 数据库，无需登录，无云端上传，充分保护用户隐私。2. 技术栈层级技术说明前端框架Next.js 14+ (App Router)全栈框架，前后端统一语言TypeScript全量类型安全样式Tailwind CSS + shadcn/ui组件库 + 原子化样式数据库SQLite + better-sqlite3本地文件数据库，隐私优先AI 接入通义千问 / DeepSeek / 自定义多模型适配层图表Recharts情感趋势、标签分布可视化运行环境纯本地，无需登录数据无云端上传3. 项目目录结构项目采用 Next.js App Router 模式，目录结构如下：diarai/├── src/│   ├── app/│   │   ├── page.tsx               # 首页 / 仪表盘│   │   ├── write/page.tsx         # 写日记（AI 引导）│   │   ├── diary/[id]/page.tsx    # 日记详情│   │   ├── stats/page.tsx         # 统计分析│   │   ├── profile/page.tsx       # 个性画像│   │   ├── tags/page.tsx          # 标签管理│   │   └── settings/page.tsx      # 设置（API Key、模型选择）│   ├── api/│   │   ├── diary/route.ts         # 日记 CRUD│   │   ├── ai/analyze/route.ts    # AI 分析（缓存）│   │   ├── ai/guide/route.ts      # AI 引导写作（流式 SSE）│   │   ├── tags/route.ts          # 标签 CRUD│   │   └── stats/route.ts         # 统计聚合查询│   ├── components/│   │   ├── layout/                # 侧边栏、顶部导航│   │   ├── diary/                 # 日记卡片、Markdown 编辑器│   │   ├── charts/                # 情感折线图、标签饼图、热力图│   │   └── ui/                    # shadcn 基础组件│   ├── lib/│   │   ├── db.ts                  # SQLite 单例连接│   │   ├── ai.ts                  # 多模型适配封装│   │   └── utils.ts               # 工具函数│   └── types/index.ts             # 全局类型定义└── data/diarai.db                 # SQLite 数据库文件4. 数据库 Schema项目共使用 4 张表：4.1 entries — 日记主表字段类型说明idTEXT PRIMARY KEYUUID，日记唯一标识符dateTEXT NOT NULL日期，格式 YYYY-MM-DDtitleTEXT日记标题（可空）contentTEXT NOT NULL日记正文，Markdown 格式mood_scoreINTEGER1~10 情绪评分created_atTEXT NOT NULL创建时间 ISO 8601updated_atTEXT NOT NULL更新时间 ISO 86014.2 tags — 标签表字段类型说明idTEXT PRIMARY KEYUUIDnameTEXT NOT NULL UNIQUE标签名称，全局唯一typeTEXT NOT NULL&apos;emotion&apos; AI 功能模块6.1 引导写作（流式 SSE）用户选择“今天发生了什么”， AI 通过追问引导用户深入写作。对话式引导，最终生成完整日记草稿。技术实现为 Server-Sent Events 流式输出，打字机效果。6.2 AI 一键分析对已有日记自动完成以下五件事：情感识别（主情绪 + 强度）关键词提取（5~8个）标签建议（从现有标签库中推荐）日记摘要（50字内）温和回应 / 建议分析结果缓存到 ai_analyses 表，同日记 + 同模型不重复调用 API。6.3 个性画像基于近30 / 90 天的日记和标签数据，AI 生成文字画像。覆盖维度包括：情绪基调、关注领域、成长轨迹、写作习惯。配合雷达图展示五维意义标签占比。6.4 多模型适配层type ModelProvider = &quot;qwen&quot; | &quot;deepseek&quot; | &quot;custom&quot;interface AIConfig {  provider: ModelProvider  apiKey: string  baseUrl?: string    // 自定义模型用  model?: string      // 具体模型版本}// 调用统一封装async function callAI(prompt: string, config: AIConfig): Promise&lt;string&gt;async function* streamAI(prompt: string, config: AIConfig): AsyncGenerator&lt;string&gt;7. 页面功能详述页面路由核心功能首页/今日写作入口、近期情绪趋势迷你折线图、快速统计（总篇数/连续天数/本月标签数）写日记/write富文本编辑器 + AI 引导对话 + 文本导入 + 情绪滑块 + 手动标签选择 + AI 分析日记列表/diary按日期分组、标签筛选、情绪筛选、关键词搜索、Markdown 预览、导出统计分析/stats情绪折线图（月/周）、标签饼图、写作频率热力图、字数统计个性画像/profileAI 文字画像 + 五维雷达图 + 高频词云标签管理/tags增删改查、合并标签、标签云可视化、使用频次排序设置/settingsAPI Key 配置、默认模型切换、本地数据导出（JSON/Markdown）、主题切换8.
