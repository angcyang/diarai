### 6. 数据缓存策略 🚀

**功能特性：**
- 30秒缓存有效期
- 避免频繁读取localStorage
- 缓存自动更新机制
- 手动清除缓存功能
- 性能优化

**缓存对象：**
```javascript
state.cache = {
    diaries: null,
    lastUpdate: 0,
    isValid: function() {
        return this.diaries !== null && (Date.now() - this.lastUpdate) < 30000;
    }
}
```

## 📝 使用说明

### AI对话引导
1. 选择"AI智能写作"模式
2. 点击"开始对话"
3. 依次回答AI的问题
4. 点击"结束对话并生成日记"
5. 查看和编辑生成的日记
6. 保存日记

### 添加示例日记
1. 选择"添加示例日记"模式
2. 浏览7篇精选示例
3. 点击"添加到日记库"
4. 示例将作为正常日记出现在日记库中

### 数据导出
1. 进入设置页面
2. 点击"导出数据"
3. 选择导出格式（JSON/Markdown/TXT/Word）
4. 文件自动下载

### 快捷键
- `Ctrl+S` - 保存日记
- `Ctrl+N` - 清空编辑器
- `Ctrl+Enter` - 发送对话回答
