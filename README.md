# AI智能日记应用

一个功能完整的AI驱动日记应用，支持AI引导写作、情绪分析和个人报告生成。

## 功能特性

### 1. 写日记主页面
- ✍️ 自由撰写日记内容
- 🤖 AI引导写作灵感（随机生成写作建议）
- 😊 情绪选择器（6种情绪类型）
- 🎭 实时情绪分析（基于关键词检测）
- 💾 本地存储保存日记

### 2. 日记库
- 📚 日记列表展示
- 🔍 关键词搜索功能
- 🎯 情绪筛选功能
- 👁️ 查看日记详情
- 🗑️ 删除日记

### 3. 个人分析
- 📊 数据统计（总日记数、连续记录天数、总字数）
- 📈 情绪趋势图表可视化
- 🎭 主要情绪分析
- 📋 AI生成个人报告
- 💡 个性化建议

### 4. 设置页面
- 🤖 多AI服务支持（OpenAI、Claude、Gemini、本地模型）
- 🔑 API Key配置
- 🔄 模型选择
- 🔒 隐私设置（加密存储、云端备份）
- 💾 数据导出功能
- 🗑️ 数据清除功能

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动应用
```bash
npm start
```

### 访问应用
在浏览器中打开：`http://localhost:3000`

## 技术栈

- **前端**：纯 HTML5 + CSS3 + JavaScript（ES6+）
- **后端**：Node.js + Express
- **存储**：localStorage（本地存储）
- **设计**：响应式设计，支持桌面和移动端

## 功能说明

### 写日记
1. 点击"获取写作灵感"获取AI写作建议
2. 在标题框输入日记标题
3. 在文本框输入日记内容
4. 选择当前心情
5. 点击"分析情绪"查看AI分析结果
6. 点击"保存日记"保存

### 日记库
1. 查看所有日记列表
2. 使用搜索框搜索日记
3. 使用情绪筛选器过滤日记
4. 点击"查看"查看日记详情
5. 点击"删除"删除日记

### 个人分析
1. 查看统计数据
2. 查看情绪趋势图表
3. 点击"生成个人报告"获取AI分析报告

### 设置
1. 选择AI服务提供商
2. 输入API Key（本地模型无需）
3. 选择AI模型
4. 配置隐私设置
5. 导出或清除数据

## 数据存储

所有数据存储在浏览器的localStorage中，包括：
- 日记内容
- 用户设置
- 情绪记录

## 扩展功能

当前版本使用模拟的AI功能，可以扩展：
- 集成真实的AI API（OpenAI、Claude等）
- 添加云端同步功能
- 增强情绪分析算法
- 添加更多图表类型
- 支持富文本编辑
- 添加图片上传功能
