# AI智能日记应用

一个功能完整的AI驱动日记应用，支持AI引导写作、情绪分析和个人报告生成。

## 功能特性

### 1. 写日记主页面
- ✍️ 自由撰写日记内容
- 🤖 AI引导写作灵感（随机生成写作建议）
- 😊 情绪选择器（6种情绪类型）
- 🎭 实时情绪分析（基于关键词检测）
- 💾 本地存储保存日记

### 2. 日记库
- 📚 日记列表展示
- 🔍 关键词搜索功能
- 🎯 情绪筛选功能
- 👁️ 查看日记详情
- 🗑️ 删除日记

### 3. 个人分析
- 📊 数据统计（总日记数、连续记录天数、总字数）
- 📈 情绪趋势图表可视化
- 🎭 主要情绪分析
- 📋 AI生成个人报告
- 💡 个性化建议

### 4. 设置页面
- 🤖 多AI服务支持（OpenAI、Claude、Gemini、本地模型）
- 🔑 API Key配置
- 🔄 模型选择
- 🔒 隐私设置（加密存储、云端备份）
- 💾 数据导出功能
- 🗑️ 数据清除功能

# DiarAI - AI 日记分析工具

一款面向个人用户的 AI 日记分析工具，支持 AI 引导写作、情感趋势分析、双向标签体系与个性画像生成。

## ✨ 功能特性

- **🤖 AI 引导写作** - 通过对话式引导，深入挖掘你的内心故事
- **😊 情感分析** - 自动识别日记情绪，生成情绪趋势图
- **🏷️ 双向标签体系** - 情感标签（7种）+ 意义标签（5类）
- **📊 个性画像** - 基于日记数据，AI 生成你的性格画像
- **📈 数据可视化** - 情绪折线图、标签分布、热力图等
- **🔒 隐私优先** - 本地 SQLite 存储，无需登录，无云端上传

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0
- npm >= 9.0

### 安装

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env.local
# 编辑 .env.local 填入你的 API Key
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 生产构建

```bash
npm run build
npm start
```

## 🔧 配置说明

### AI API Key 配置

支持三种 AI 提供商：

1. **通义千问**（默认）
   - 获取地址：https://dashscope.console.aliyun.com/
   - 模型推荐：qwen-turbo（快速）或 qwen-plus（标准）

2. **DeepSeek**
   - 获取地址：https://platform.deepseek.com/
   - 模型推荐：deepseek-chat

3. **自定义**
   - 支持任何 OpenAI 兼容 API
   - 需要配置 API 地址和模型名称
